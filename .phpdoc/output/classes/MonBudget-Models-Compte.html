Class Compte

Modèle de gestion des comptes bancaires.

Représente un compte bancaire appartenant à un utilisateur et lié à une banque. Gère les différents types de comptes (courant, épargne, livret, etc.), les soldes (initial et actuel), les devises et les opérations de recalcul.

recalculerSolde()

Compte.php : 391

Recalcule et met à jour le solde actuel basé sur toutes les transactions

public static recalculerSolde(int $id) : bool

Approche sécurisée avec ABS() : fonctionne que les débits soient stockés en positif ou négatif dans la base de données.

Formule : solde_actuel = solde_initial + SUM(crédits) - SUM(débits)

Cette méthode garantit la cohérence entre le solde stocké et les transactions, notamment après import de fichiers bancaires ou corrections manuelles.

Parameters
$id : int

ID du compte à recalculer

Tags
example

// Après import de transactions if (Compte::recalculerSolde($compteId)) { echo "Solde recalculé avec succès"; }

see

Return values
bool —

True si le recalcul a réussi, false si le compte n'existe pas
