Database.php : 19

Gestionnaire de connexion et d'opérations sur la base de données

Classe Singleton fournissant une connexion PDO unique et des méthodes utilitaires pour les opérations CRUD : select, insert, update, delete. Gère automatiquement les requêtes préparées pour la sécurité.
